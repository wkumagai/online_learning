# 注文ログディレクトリ

このディレクトリには、取引システムによって生成された注文の記録が保存されます。

## ファイル形式
- CSV: 構造化された注文データ
- JSON: 詳細な注文情報とメタデータ
- LOG: テキスト形式のログ

## 命名規則
mode_strategy_YYYYMMDD.extension

例：
- paper_moving_average_20230101.csv
- real_rsi_strategy_20230215.json
- backtest_ml_model_20230320.log

## モード
- paper: ペーパートレード（仮想取引）
- real: 実取引
- backtest: バックテスト

## 記録内容
- 注文ID
- タイムスタンプ
- 銘柄/ティッカー
- 注文タイプ（市場、指値、逆指値など）
- 方向（買い/売り）
- 数量
- 価格
- 手数料
- 状態（送信済、約定、キャンセルなど）
- 戦略ID
- 実行環境

## 注意事項
- 実取引の注文ログは重要な記録であり、バックアップを取ってください
- 注文ログは監査とパフォーマンス分析に使用されます
- 大規模なログファイルは定期的にアーカイブしてください
- 機密情報（アカウント詳細など）は適切に保護してください
